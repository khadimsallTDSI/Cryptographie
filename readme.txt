Le projet Cryptojava est un projet de j2EE .
Dans ce dernier on peut faire de la generation de Cle Secrete , Symetrique/Asymetrique,
de faire le chiffrement symétrique ou asymétrique en utilisant les clés précédentes.
Le chiffrement peut être fait a travers un text ou un fichier.txt.
On peut aussi signer  un message et verifier sa signature.
A travers cette application java/j2EE on peut retrouver l'onglet historique ou on peut 
retrouver les clés symétrique/asymetrique deja generes.
Y'a également une base de données
